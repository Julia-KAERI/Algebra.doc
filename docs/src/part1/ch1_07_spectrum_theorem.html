여기서는 복수체와 실수체에서의 스펙트럼 정리를 알아본다. 아마 선형대수학에서 가장 중요한 것을 꼽으라면 스펙트럼 정리일 것이다.


7.1 정규연산자와 정규행렬

7.1.1 정규 연산자와 행렬

정의 7.1 (정규 행렬) 내적벡터공간에서 정의된 \(L\in \mathcal{L}(U,\,V)\) 가 \(LL^\ast = L^\ast L\) 일 때 \(L\) 을 정규연산자(normal operator) 라고 한다. 마찬가지로 정사각 행렬 \(\boldsymbol{L}\in \mathcal{M}_{n \times n}(\mathbb{F})\) 이 \(\boldsymbol{LL}^\ast = \boldsymbol{L}^\ast \boldsymbol{L}\) 일 때 \(\boldsymbol{L}\) 을 정규 행렬(normal matrix) 라 한다.


다음은 자연스럽다.


명제 7.1 에르미트 연산자는 정규연산자이다. 또한 에르미트 행렬은 정규 행렬이다.


명제 7.2 \(M\in \mathcal{L}(V)\) 가 정규연산자일 필요충분인 조건은 모든 \(v\in V\) 에 대해 \(\|Mv\| = \|M^\ast v\|\) 인 것이다.

(증명). \[ \begin{aligned} M \text{ is normal} &\iff \langle (M^\ast M - MM^\ast)v ,\, v \rangle = 0 \\[0.3em] &\iff \langle M^\ast M v,\, v \rangle = \langle MM^\ast v ,\, v\rangle \\ & \iff \|Mv\|^2 = \|M^\ast v\|^2 \end{aligned} \]


명제 7.3 \(\mathbb{F}\)-벡터공간 \(V\) 에 대해 \(M\in \mathcal{L}(V)\) 가 정규연산자일 때 다음이 성립한다.

 (\(a\)) \(\ker (M) = \ker (M^\ast)\).

 (\(b\)) \(\text{im}(M) = \text{im} (M^\ast)\).

 (\(c\)) \(V = \ker (M) \oplus \text{im}(M)\).

 (\(d\)) \(c\in \mathbb{F}\) 일 때 \(M-c I\) 도 정규연산자이다.

(증명). 전체적으로 정리 6.1 를 참고하라.

(\(a\)) 명제 7.2

(\(b\)) \(\text{im}(M) = (\ker (M^\ast))^\perp = (\ker (M))^\perp = \text{im}(M^\ast)\).

(\(c\)) \(V=(\ker (M)) \oplus (\ker (M))^\perp = (\ker (M)) \oplus (\ker (M^\ast))^\perp = \ker (M)\oplus \text{im}(M)\)

(\(d\))

\[ \begin{aligned} (M-cI)(M-cI)^\ast &= (M-cI)(M^\ast - \overline{c}I) = MM^\ast - \overline{c}L + cM^\ast -|c|^2 \\[0.3em] &=M^\ast M - \overline{c}M + cM^\ast -|c|^2 = (M-cI)^\ast (M-cI) \end{aligned} \]


명제 7.4 \(M\) 이 정규연산자이면 \(M^\ast\) 와 같은 고유벡터를 가지며 이에 대한 고유값은 서로 복소 공액이다.

(증명). \(M-cI\) 가 정규연산자이므로 명제 7.2 로부터 \(0 = \|(M-cI)v\| = \|(M-cI)^\ast v\|\). \(\square\)


명제 7.5 정규연산자 \(M\) 의 서로 다른 고유값을 갖는 두 고유벡터는 직교한다.

(증명). \(M\) 의 고유값 \(c_1,\,c_2\) 에 대한 고유벡터 \(v_1,\,v_2\) 를 생각하자. 이 때 \(c_1\ne c_2\) 이다.

\[ \begin{aligned} (c_1-c_2)\langle v_1,\,v_2\rangle &= \langle Mv_1,\,v_2\rangle - \langle v_1,\, M^\ast v_2 \rangle = \langle Mv_1,\,v_2\rangle - \langle Mv_1,\, v_2 \rangle = 0 \end{aligned} \]

이며 \(c_1 \ne c_2\) 이므로 \(\langle v_1,\,v_2\rangle = 0\) 이다. \(\square\)


명제 7.6 정규행렬의 상삼각행렬꼴은 대각행렬이다.

(증명). 우리는 슈어 정리(정리 5.2) 로부터 모든 복소행렬이 어떤 정규직교기저에서 상삼각행렬꼴로 나타남을 안다. 이 상삼각행렬을 \(\boldsymbol{A}\) 라 하면 정규직교기저의 행렬형식은 표준기저 \((\boldsymbol{e}_i)\) 가 된다. \(\boldsymbol{Ae}_j\) 는 \(\boldsymbol{A}\) 의 \(j\) 번째 열벡터 \(\boldsymbol{A}_{:j}\) 이며, \(\boldsymbol{A}\) 가 상삼각행렬 이므로 \(\|\boldsymbol{Ae}_j\|^2 = \sum_{i=1}^j |A_{ij}|^2\) 이다. \(\boldsymbol{A}^\ast\) 는 하삼각행렬이며, 땨라서

\[ \|\boldsymbol{A}^\ast\boldsymbol{e}_j\|^2= \sum_{i=j}^n \left|(\boldsymbol{A}^\ast)_{ij}\right|^2 = \sum_{i=j}^n |A_{ji}|^2 \]

이다. 정규행렬의 정의에 의해 \(\|\boldsymbol{Ae}_j\|^2 = \|\boldsymbol{A}^\ast \boldsymbol{e}_j\|^2\) 이어야 한다. \(j=1\) 인 경우

\[ |A_{11}|^2 = |A_{11}|^2 + \cdots + |A_{1n}|^2 \]

이므로 \(A_{12} = \cdots = A_{1n}=0\) 이며, \(j=2\) 인 경우

\[ |A_{12}|^2 + |A_{22}|^2 = |A_{22}|^2 + \cdots + |A_{2n}|^2 \]

인데 \(j=1\) 인 경우에서 \(A_{12}=0\) 임을 보였으므로 \(A_{23}= \cdots = A_{2n}=0\) 임을 안다. 이것을 계속 반복해보면 \(\boldsymbol{A}\) 가 대각행렬임을 알 수 있다. \(\square\)


정리 7.1 \(\mathbb{C}\) 벡터공간 \(V\) 에서 정의된 \(L\in \mathcal{L}(V)\) 에 대해 다음은 동치이다.

 (\(1\)) \(L\) 은 정규연산자이다.

 (\(2\)) 어떤 서로 commute 한 에르미트 연산자 \(A,\,B \in \mathcal{L}(V)\) 에 대해 \(L=A+iB\) 이다.

(증명). (\(1\implies 2\)) \(L\) 이 정규연산자라고 하자. \(A:=\dfrac{L+L^\ast}{2},\, B:= \dfrac{L-L^\ast}{2i}\) 라고 하면 \(L = A+iB\) 이며 \(A,\,B\) 는 에르미트 연산자이다. 또한

\[ \begin{aligned} AB-BA &= \dfrac{LL-LL^\ast + L^\ast L - L^\ast L^\ast - LL + L^\ast L - LL^\ast + L^\ast L^\ast}{4i} \\[0.3em] &= \dfrac{L^\ast L - LL^\ast}{2i} = 0 \end{aligned} \]

이므로 \(A\) 와 \(B\) 는 commute 하다.

(\(2\implies 1\)) \(LL^\ast - L^\ast L = (A+iB)(A-iB)- (A-iB)(A+iB)= i(BA-AB)=0\) 이므로 \(L\) 은 정규연산자이다. \(\square\)

연습문제 7.4 (Axler 7.A.12) \(\boldsymbol{A}\in \mathcal{M}_{n \times n}(\mathbb{F})\) 가 정규행렬이고 \(3\) 과 \(4\) 를 고유값으로 가진다고 하자. 이 때 \(\boldsymbol{u}\in \mathcal{M}_n(\mathbb{F})\), \(\|\boldsymbol{u}\|=\sqrt{2}\), \(\|\boldsymbol{Au}\|=5\) 인 \(\boldsymbol{u}\) 가 존재함을 보여라.

(해답). \(3\) 을 고유값으로 가지고 크기가 \(1\) 인 벡터를 \(\boldsymbol{u}_1\) 이라 하고 \(4\) 를 유값으로 가지고 크기가 \(1\) 인 벡터를 \(\boldsymbol{u}_2\) 라고 하자. 서로 다른 고유값을 가진 정규벡터의 고유벡터이므로 \(\boldsymbol{u}_1\) 과 \(\boldsymbol{u}_2\) 는 직교한다. \(\boldsymbol{u}=\boldsymbol{u}_1+\boldsymbol{u}_2\) 라고 하면, \(\|\boldsymbol{u}\|=2\) 이며, \(\|\boldsymbol{Au}\|^2 = 5\) 이다.


연습문제 7.5 (Axler 7.A.15) \(\mathbb{F}\)-내적벡터공간 \(V\) 와 \(u,\,x\in V\) 에 대해 \(T\in \mathcal{L}(V)\) 를

\[ T(v) = \langle v,\, u\rangle x \]

로 정의하자. 이 때 다음이 성립함을 보여라.

  (\(1\)) \(\mathbb{F}=\mathbb{R}\) 일 때 \(T\) 가 에르미트 연산자일 필요충분조건은 \(u,\,x\) 가 선형종속인 것이다.

  (\(2\)) \(T\) 가 정규연산자일 필요충분조건은 \(u,\,x\) 가 선형종속인 것이다.

(해답). \(v,\,w\in V\) 에 대해 \(\langle T^\ast(v), w\rangle = \langle v, T(w)\rangle\) 이다. 따라서,

\[ \langle T^\ast (v),\, w\rangle = \langle u, w\rangle \langle v, x\rangle \]

로부터 \(T^\ast (v) = \langle v, x\rangle u\) 임을 안다.

(\(1\)) \(T\) 가 에르미트 연산자라면 \(T=T^\ast\) 이어야 하므로,

\[ 0=(T-T^\ast)(v) = \langle v, u\rangle x - \langle v, x\rangle u \]

으로부터 \(x= \dfrac{\langle v, u\rangle}{\langle v, x\rangle}u\) 이어야 한다. 즉 \(u\) 와 \(x\) 는 선형 종속이어야 한다. 이제 \(u\) 와 \(x\) 가 선형종속이라고 하자. 즉 \(u=cx,\, c\in \mathbb{R}\) 이라고 하자.

\[ (T-T^\ast)(v) = c\langle v, x\rangle x - c \langle v, x \rangle x = 0 \]

이므로 \(T=T^\ast\) 이다.

(\(2\)) \(T\) 가 정규연산자라고 하자. \(TT^\ast - T^\ast T=0\) 이므로,

\[ \begin{aligned} 0 &= (TT^\ast - T^\ast T)(v) = T(\langle v, x \rangle u) - T^\ast (\langle v, u\rangle x) \\ & = \langle v, x \rangle \|u\|^2 x - \langle v, u\rangle \|x\|^2 u \end{aligned} \]

이다. 즉 \(x = \dfrac{\langle v, x\rangle \|u\|^2}{\langle v, u\rangle \|x\|^2} u\) 이어야 하므로 선형종속이다. 이제 \(u=cx,\, c\in \mathbb{F}\) 라고 하자.

\[ (TT^\ast - T^\ast T)(v) = T(c\langle v, x \rangle x) - T^\ast (c\langle v, x\rangle x) = c^2\langle v, x \rangle^2 x - c^2\langle v, x \rangle^2 x = 0 \]

이므로 \(T\) 는 정규연산자이다.


연습문제 7.6 (Axler 7.A.16) \(\boldsymbol{A}\in \mathcal{M}_{n \times n}(\mathbb{F})\) 가 정규연산자일 때 다음을 보이시오.

  (\(1\)) \(\text{im}(\boldsymbol{A}) = \text{im}(\boldsymbol{A}^\ast)\).

  (\(2\)) 양의 정수 \(k\) 에 대해 \(\ker (\boldsymbol{A}^k) = \ker (\boldsymbol{A})\) 이며 \(\text{im}(\boldsymbol{A}^k) = \text{im}(\boldsymbol{A})\).

(해답). (\(1\)) 명제 7.2 로부터 \(v\in \ker (\boldsymbol{A})\) 이면 \(0=\|\boldsymbol{Av}\|=\|\boldsymbol{A}^\ast \boldsymbol{v}\|\) 이므로 \(\boldsymbol{v} \in \ker (\boldsymbol{A}^\ast)\) 임을 안다. \((\boldsymbol{A}^\ast)^\ast = \boldsymbol{A}\) 이므로 \(\ker (\boldsymbol{A})= \ker(\boldsymbol{A}^\ast)\) 이다. 또한 정리 6.1 로부터

\[ \text{im}(\boldsymbol{A}) = (\ker (\boldsymbol{A}^\ast))^\perp = (\ker (\boldsymbol{A}))^\perp = \text{im}(\boldsymbol{A}^\ast) \]

임을 안다.

(\(2\)) \(k\) 에 대한 induction 을 사용한다. \(\ker (\boldsymbol{A})\subseteq \ker(\boldsymbol{A}^k)\) 임은 자명하므로 \(\ker(\boldsymbol{A}^k) \subseteq \ker(\boldsymbol{A})\) 만 보이면 된다. \(\boldsymbol{A}\) 가 정규행렬이라면 양의 정수 \(k\) 에 대해 \((\boldsymbol{A}^\ast)^k\boldsymbol{A}^k = (\boldsymbol{A}^\ast\boldsymbol{A})^k\) 임은 쉽게 보일 수 있다. 또한

\[ \begin{aligned} \|\boldsymbol{A}^2\boldsymbol{v}\|= \langle \boldsymbol{A}^2 \boldsymbol{v}, \boldsymbol{A}^2 \boldsymbol{v} \rangle = \langle \boldsymbol{A}^\ast \boldsymbol{Av},\boldsymbol{A}^\ast \boldsymbol{Av}\rangle = \|\boldsymbol{A}^\ast \boldsymbol{Av}\|^2 \end{aligned} \]

이므로 \(\ker (\boldsymbol{A}^2) = \ker (\boldsymbol{A}^\ast \boldsymbol{A})\) 이며,

\[ \boldsymbol{v}\in \ker (\boldsymbol{A}^\ast\boldsymbol{A}) \implies \langle \boldsymbol{A}^\ast \boldsymbol{Av}, \boldsymbol{v}\rangle = 0 \implies \|\boldsymbol{Av}\|^2=0 \]

이므로 \(\ker (\boldsymbol{A}^2)\subseteq \ker(\boldsymbol{A})\) 임을 안다. \(\ker(\boldsymbol{A}) \subseteq \ker(\boldsymbol{A}^2)\) 임은 자명하므로 \(\ker(\boldsymbol{A}) = \ker(\boldsymbol{A}^2)\) 이다. 이제 \(\ker(\boldsymbol{A}^k) = \ker(\boldsymbol{A})\) 임을 가정하자.

\[ \boldsymbol{v}\in \ker (\boldsymbol{A}^{k+1}) \implies \boldsymbol{A}^k(\boldsymbol{A}\boldsymbol{v}) = \boldsymbol{0} \implies \boldsymbol{Av} \in \ker (\boldsymbol{A}^k) \implies \boldsymbol{Av} \in \ker (\boldsymbol{A}) \]

이므로 \(\boldsymbol{v}\in \ker(\boldsymbol{A}^2) = \ker(\boldsymbol{A})\) 이다.

이제 \(\text{im}(\boldsymbol{A}^k) = \text{im}(\boldsymbol{A})\) 임을 보이자. 우리는 앞서 \(\ker (\boldsymbol{A}) = \ker (\boldsymbol{A}^\ast)\) 임을 보였다. \(\boldsymbol{A}\) 가 정규행렬이라면 \(\boldsymbol{A}^\ast\) 도 정규행렬이므로

\[ \ker (\boldsymbol{A}^k) = \ker (\boldsymbol{A}) =\ker (\boldsymbol{A}^\ast) = \ker ((\boldsymbol{A}^\ast)^k) = \ker ((\boldsymbol{A}^k)^\ast) \]

이다. 따라서 \[ \text{im}(\boldsymbol{A}^k) = (\ker ((\boldsymbol{A}^k)^\ast))^\perp = (\ker (\boldsymbol{A}^\ast)) = \text{im}(\boldsymbol{A}) \]

이다.


연습문제 7.7 (Axler 7.A.18) \(\boldsymbol{A}\in \mathcal{M}_{n\times n}(\mathbb{F})\) 가 표준기저 \((\boldsymbol{e}_1, \ldots,\,\boldsymbol{e}_n)\) 에 대해 \(\|\boldsymbol{Ae}_i\| = \|\boldsymbol{A}^\ast \boldsymbol{e}_i\|\) \((i=1,\ldots,\,n)\) 이면 \(\boldsymbol{A}\) 가 정규행렬인가?

(해답). (이게 맞다면 텍스트에 명제로 나왔겠지?)

\(\boldsymbol{A}= \left[\begin{array}{rr} 1 & -1 \\ 1 & -1\end{array}\right]\) 에 대해 \(\|\boldsymbol{Ae}_1\| = \|\boldsymbol{Ae}_2\| = \sqrt{2}\) 이지만 \(\boldsymbol{AA}^\ast - \boldsymbol{A}^\ast\boldsymbol{A} = \left[\begin{array}{rr} 0 & 4 \\ 4 & 0\end{array}\right] \ne \boldsymbol{0}\) 이다.


연습문제 7.8 (Axler 7.B.1) 대칭행렬이 아니면서 행렬의 고유벡터로 이루어진 기저를 갖도록 하는 정사각행렬이 존재하는가?

(해답). \(\boldsymbol{A} = \begin{bmatrix} 1 & 1 & 0 \\ 0 & 2 & 0 \\ 0 & 0 & 3\end{bmatrix}\) 는 \(\boldsymbol{Ae}_1 = \boldsymbol{e}_1\), \(\boldsymbol{A}(\boldsymbol{e}_1 + \boldsymbol{e}_2) = 2(\boldsymbol{e}_1 + \boldsymbol{e}_2)\), \(\boldsymbol{Ae}_3 = 3\boldsymbol{e}_3\) 인데 대칭행렬이 아니다.

스펙트럼 정리가 이야기하는 것은 복소수체에서의 정규행렬, 실수체에서의 대칭행렬의 경우 행렬의 고유벡터로 정규직교기저를 구성할 수 있다는 것이다. 위의 문제의 경우는 대칭행렬이 아니므로 고유벡터로 정규직교기저를 구성할 수 없지만 기저는 구성 할 수 있다는 것이다.


연습문제 7.9 (Axler 7.B.4) \(\boldsymbol{A}\in \mathcal{M}_{n \times n}(\mathbb{C})\) 일 때 \(\boldsymbol{A}\) 가 정규행렬인것의 필요충분조건은 서로 다른 고유값을 갖는 고유벡터의 모든 쌍은 서로 직교하며 \(\mathcal{M}_n(\mathbb{C})\) 는 모든 \(\boldsymbol{A}\) 의 고유값에 대한 고유공간의 직합과 같다는 것임을 보여라.

(해답). 정리 7.2


연습문제 7.10 (Axler 7.B.5) \(\boldsymbol{A}\in \mathcal{M}_{n \times n}(\mathbb{R})\) 일 때 \(\boldsymbol{A}\) 가 대칭행렬인것의 필요충분조건은 서로 다른 고유값을 갖는 고유벡터의 모든 쌍은 서로 직교하며 \(\mathcal{M}_n(\mathbb{R})\) 는 모든 \(\boldsymbol{A}\) 의 고유값에 대한 고유공간의 직합과 같다는 것임을 보여라.

(해답). 정리 7.3


7.2 복소수체(\(\mathbb{C}\)) 에서 정의된 행렬에 대한 스펙트럼 정리

정리 7.2 (복소수체에서 정의된 행렬에 대한 스펙트럼 정리) \(\boldsymbol{A} \in \mathcal{M}_{n \times n}(\mathbb{C})\) 행렬에 대해 다음 세 명제는 동치이다.

 (\(1\)) \(\boldsymbol{A}\) 는 정규행렬이다.

 (\(2\)) \(\boldsymbol{A}\) 는 unitary 변환에 의해 대각행렬로 표현된다.

 (\(3\)) \(\boldsymbol{A}\) 의 고유벡터로 \(\mathcal{M}_{n }(\mathbb{C})\) 의 정규직교기저를 구성 할 수 있다.

(증명). (\(1 \implies 2\)) 는 명제 7.6 에서 보였다.

(\(2\implies 3\)) \(\boldsymbol{A}\) 가 unitary 행렬 \(\boldsymbol{P}\) 에 의해 대각행렬로 변환된다고 하자. 즉 \(\boldsymbol{P}^{-1}\boldsymbol{AP}=\boldsymbol{D}\) 는 대각행렬이다. 대각행렬이 주어졌을 때, 표준정규직교벡터 \(\{\boldsymbol{e}_1,\ldots,\,\boldsymbol{e}_n\}\) 는 \(\lambda_i = D_{ii}\) 를 고유값으로 갖는 대각행렬의 고유벡터이다. 여기서

\[ \boldsymbol{APe}_i = \boldsymbol{P}\boldsymbol{P}^{-1}\boldsymbol{APe}_i= \boldsymbol{PDe}_i=\lambda_i \boldsymbol{Pe}_i \]

이므로 \(\boldsymbol{Pe}_i\) 는 \(\boldsymbol{A}\) 의 고유벡터이다. \(\boldsymbol{P}\) 가 linear isometry 이므로 \(\{\boldsymbol{Pe}_1,\ldots,\,\boldsymbol{Pe}_n\}\) 은 \(\boldsymbol{A}\) 의 고유벡터로 이루어진 \(\mathcal{M}_n(\mathbb{C})\) 의 정규직교기저이다.

(\(3. \implies 1.\)) \(\{\boldsymbol{e}_1,\ldots,\,\boldsymbol{e}_n\}\) 이 \(\boldsymbol{A}\) 의 고유벡터로 이루어진 \(\mathcal{M}_{n }(\mathbb{C})\) 의 정규직교기저라 하자. 이 때 \(\boldsymbol{u}_i\) 의 고유값은 \(\lambda_i\) 라 하자.

\[ \langle \boldsymbol{A}^\ast \boldsymbol{e}_i,\,\boldsymbol{e}_j\rangle = \langle \boldsymbol{e}_i,\, \boldsymbol{Ae}_j\rangle = \overline{\lambda_j} \delta_{ij} \]

이다. 즉,

\[ \boldsymbol{A}^\ast \boldsymbol{e}_i = \overline{\lambda_i} \boldsymbol{e}_i \]

이다. \(\boldsymbol{u} = \sum_i u_i \boldsymbol{e}_i\) 에 대해,

\[ \begin{aligned} \|\boldsymbol{A}^\ast \boldsymbol{u} \|^2 &= \left\langle \boldsymbol{A}^\ast \left(\sum_i u_i \boldsymbol{e}_i \right),\, \boldsymbol{A}^\ast \left(\sum_j u_j \boldsymbol{e}_j \right) \right\rangle \\ &=\sum_{i,\,j} u_i \overline{u_j} \left\langle \boldsymbol{A}^\ast\boldsymbol{e}_i,\, \boldsymbol{A}^\ast \boldsymbol{e}_j\right\rangle \\ &= \sum_{i} |u_i|^2 |\lambda_i|^2 \\ &= \|\boldsymbol{A}\boldsymbol{u}\|^2 \end{aligned} \]

이므로 \(\boldsymbol{A}\) 는 정규행렬이다. \(\square\)


정규행렬은 고유벡터로 이루어진 정규직교기저가 존재하며, 따라서 이 기저에서 대각행렬로 변환된다는 좋은 성질을 가진다. 에르미트 행렬은 정규행렬이므로 이 성질을 공유하며 더하여 고유값이 항상 실수이다.


연습문제 7.11 복소 정사각 행렬 \(\boldsymbol{A}\in \mathcal{M}_{n \times n}(\mathbb{C})\) 이 정규행렬이며 그 고유값이 모두 실수이면 \(\boldsymbol{A}\) 는 에르미트 행렬임을 보여라.

(해답). \(\{\boldsymbol{e}_1,\ldots,\,\boldsymbol{e}_n\}\) 이 \(\boldsymbol{A}\) 의 고유벡터로 이루어진 \(\mathcal{M}_{n }(\mathbb{C})\) 의 정규직교기저이고 \(\boldsymbol{e}_i\) 의 고유값은 \(\lambda_i\) 라 하자. 명제 6.5 에서 보았듯이 임의의 \(\boldsymbol{u} \in \mathcal{M}_{n }(\mathbb{C})\) 에 대해 \(\langle \boldsymbol{Au},\boldsymbol{u}\rangle\) 이 실수이면 \(\boldsymbol{A}\) 는 에르미트 행렬이다. \(\boldsymbol{u}= \sum_i^n u_i \boldsymbol{e}_i\) 라 하면,

\[ \begin{aligned} \langle \boldsymbol{Au},\, \boldsymbol{u}\rangle = \sum_i \lambda_i |u_i|^2 \in \mathbb{R} \end{aligned} \]

이다. 즉 \(\boldsymbol{A}\) 는 에르미트 행렬이다. \(\square\)


7.3 실수체(\(\mathbb{R}\)) 에서 정의된 행렬에 대한 스펙트럼 정리

7.3.1 미리 알아야 할 것들

실수체에서 정의된 행렬 \(\boldsymbol{A}\) 가 에르미트 행렬이면 대칭행렬이며, 대칭행렬이면 에르미트 행렬이다. 실수체에서 정의된 스펙트럼 정리를 증명하기 위해서는 몇가지 미리 증명해야 할 것이 있다.


보조정리 7.1 \(\boldsymbol{A}\in \mathcal{M}_{n \times n}(\mathbb{R})\) 행렬에 대해 \(\boldsymbol{A}\) 가 대칭행렬이고 실수 \(b,\,c\) 가 \(b^2-4c <0\) 일 때

\[ \boldsymbol{A}^2+b\boldsymbol{A}+c\boldsymbol{I}_n \]

은 가역행렬이다.

(증명). 영벡터가 아닌 벡터 \(\boldsymbol{v}\in \mathbb{R}^{n}\) 에 대해

\[ \begin{aligned} \langle (\boldsymbol{A}^2+b\boldsymbol{A}+c\boldsymbol{I}_n) \boldsymbol{v},\, \boldsymbol{v} \rangle &= \langle \boldsymbol{Av},\,\boldsymbol{Av} \rangle + b\langle \boldsymbol{Av},\,\boldsymbol{v} \rangle + c\langle \boldsymbol{v},\,\boldsymbol{v} \rangle \\ &= \|\boldsymbol{Av}\|^2 + b\langle \boldsymbol{Av}, \,\boldsymbol{v}\rangle + c\langle \boldsymbol{v},\,\boldsymbol{v}\rangle \end{aligned} \]

이다. 코시-슈바르츠 부등식으로부터,

\[ |b\langle \boldsymbol{Av},\,\boldsymbol{v}\rangle | \le |b| \cdot \|\boldsymbol{Av}\|\|\boldsymbol{v}\| \]

임을 안다. 즉 \(b\langle \boldsymbol{Av},\,\boldsymbol{v}\rangle \ge -|b| \|\boldsymbol{Av}\|\|\boldsymbol{v}\|\) 이며,

\[ \begin{aligned} \langle (\boldsymbol{A}^2+b\boldsymbol{A}+c\boldsymbol{I}_n) \boldsymbol{v},\, \boldsymbol{v} \rangle & = \|\boldsymbol{Av}\|^2 + b\langle \boldsymbol{Av} ,\,\boldsymbol{v}\rangle + c\langle \boldsymbol{v},\,\boldsymbol{v}\rangle \\ & \ge \|\boldsymbol{Av}\|^2-|b|\|\boldsymbol{Av}\|\|\boldsymbol{v}\| + c\|\boldsymbol{v}\|^2 \\ &= \left(\|\boldsymbol{Av}\| - \dfrac{|b|\|\boldsymbol{v}\|}{2}\right)^2 + \left(c-\dfrac{b^2}{4}\right)\|\boldsymbol{Av}\|^2 >0 \end{aligned} \]

이다. \(\ker (\boldsymbol{A}^2+b\boldsymbol{A}+c\boldsymbol{I}_n) =\{0\}\) 이므로 \(\boldsymbol{A}^2+b\boldsymbol{A}+c\boldsymbol{I}_n\) 는 가역행렬이다. \(\square\)


보조정리 7.2 \(n\) 차 실계수 다항식 \(p(x)\) 는 1차식들과 2차식의 곱들로, 2차식의 경우 \(f(x)\) 라 했을 때 \(f(x)=0\) 을 만족하는 실수가 존재하지 않도록 아래와 같이 분해된다.

\[ p(x) = c(x-\lambda_1) \cdots (x-\lambda_m)(x^2 + b_1x + c_1)\cdots (x^2 + b_M x + c_M) \]

이 때, \(c,\,\lambda_1,\ldots,\,\lambda_m,\,b_1,\ldots,\,b_M,\,c_1,\ldots,\,c_M\) 은 실수이며, \(i=1,\ldots,\,M\) 에 대해 \(b_i^2<4c\) 이다.

(증명). \(p(x) = a_0 +a_1x + \cdots + a_nx^n,\, a_i\in \mathbb{R}\) 가 실수근 \(\lambda_1,\ldots,\,\lambda_m\) 을 가지면 \(\prod_{i=1}^m (x-\lambda_i)\) 로 우선 분해되고, 나머지는 2차식의 곱으로 분해된다는 의미이다. 대수학의 기본정리에 의해 \(n\) 차 다항식은 (중근을 중복하여 세면) \(n\) 개의 근을 가진다. 실수가 아닌 복소수 \(\lambda\) 를 근으로 가진다고 하자.

\[ p(\lambda) = a_0 + a_1 \lambda + \cdots + a_n \lambda^n = 0 \]

이며 위의 식에 켤레복소수를 취하면

\[ \overline{p(\lambda)} = a_0 + a_1 \overline{\lambda} + \cdots + a_n \overline{\lambda^n} = 0 \]

이므로 \(\overline{\lambda}\) 를 근으로 가진다. 즉 실계수 다항식 \(p(x)\) 의 근은 실수이거나 복소수와 켤레복소수의 쌍이다. 복소수를 근으로 갖기 위해서는 \(b_i^2<4c_i\) 이어야 한다. \(\square\)


7.3.2 실수체에서 정의된 행렬에 대한 스펙트럼 정리

보조정리 7.3 \(\boldsymbol{A}\in \mathcal{M}_{n \times n}(\mathbb{R})\) 이 대칭행렬이면 고유값을 가진다.

(증명). 영벡터가 아닌 \(\boldsymbol{v} \in \mathcal{M}_{n }(\mathbb{R})\) 을 생각하자. \(\{\boldsymbol{v},\, \boldsymbol{Av},\ldots,\, \boldsymbol{A}^n \boldsymbol{v}\}\) 는 포함된 벡터의 갯수가 \(n\) 보다 많으므로 선형종속이며, 따라서

\[ 0=c_0 \boldsymbol{v} + c_1 \boldsymbol{Av} + \cdots + c_n \boldsymbol{A}^n \boldsymbol{v} \]

를 만족하는 nontrivial 한 \(\{c_i\}\) 가 존재한다. 보조정리 7.2 에 의해,

\[ \begin{aligned} c_0 \boldsymbol{v} + & c_1 \boldsymbol{Av} + \cdots + c_n \boldsymbol{A}^n\boldsymbol{v} \\ &= c(\boldsymbol{A}-\lambda_1 \boldsymbol{I})\cdots (\boldsymbol{A}-\lambda_m \boldsymbol{I}) (\boldsymbol{A}^2 +b_1 \boldsymbol{A}+c_1\boldsymbol{I})\cdots (\boldsymbol{A}^2 +b_M \boldsymbol{A}+c_M\boldsymbol{I})\boldsymbol{v} \end{aligned} \]

로 분해 할 수 있다. 이 때 \(i=1,\ldots,\,M\) 에 대해 \(b_i^2<4c_i\) 이다. 보조정리 7.1 에 의해 \((\boldsymbol{A}^2 +b_i\boldsymbol{A}+c_i\boldsymbol{I})\) 꼴의 행렬은 가역이다. 그렇다면 최소한 하나의 \(\boldsymbol{A}-\lambda_j \boldsymbol{I}\) 꼴의 행렬이 가역행렬이 아니며, 이 때의 \(\lambda_j\) 가 고유값이다. 즉 \(\boldsymbol{A}\) 는 고유값을 가진다.\(\square\)


보조정리 7.4 \(\boldsymbol{A}\in \mathcal{M}_{n \times n}(\mathbb{R})\) 이 대칭행렬 이고 \(U\le \mathcal{M}_n(\mathbb{R})\) 가 \(\boldsymbol{A}\) 에 대한 불변부분공간일 때,

 (\(1\)) \(U^\perp\) 도 \(\boldsymbol{A}\)에 대한 불변부분공간이다.

 (\(2\)) \(\boldsymbol{A}|_U\) 도 대칭행렬 이다.

 (\(3\)) \(\boldsymbol{A}|_{U^\perp}\) 도 대칭행렬 이다.

(증명). (\(1\)) \(\boldsymbol{u}\in U,\, \boldsymbol{v}\in U^\perp\) 라 하자. \(\boldsymbol{Au}\in U\) 이며 \(\boldsymbol{A}\) 가 에르미트 행렬이므로,

\[ \langle \boldsymbol{Av},\, \boldsymbol{u}\rangle = \langle \boldsymbol{v},\, \boldsymbol{Au}\rangle =0 \]

이다. 따라서 \(U^\perp\) 도 \(\boldsymbol{A}\) 에 대한 불변부분공간이다.

(\(2\)) \(\boldsymbol{u}_1,\,\boldsymbol{u}_2\in U\) 에 대해,

\[ \langle \boldsymbol{A}|_U \boldsymbol{u}_1,\,\boldsymbol{u}_2\rangle = \langle \boldsymbol{Au}_1,\, \boldsymbol{u}_2 \rangle = \langle \boldsymbol{u}_1,\, \boldsymbol{Au}_2\rangle = \langle \boldsymbol{u}_1,\,\boldsymbol{A}|_U\boldsymbol{u}_2\rangle \]

이므로 \(\boldsymbol{A}|_U\) 도 대칭행렬 이다.

(\(3\)) (\(1\)), (\(2\)) 로 부터 자명하다. \(\square\)


보조정리 7.5 \(\boldsymbol{A}\in \mathcal{M}_{n \times n}(\mathbb{R})\) 가 대칭행렬이고 \(U,\,V \le \mathcal{M}_{n}(\mathbb{R})\) 가 \(\boldsymbol{A}\) 에 대한 불변부분공간이며 \(\mathcal{M}_{n }(\mathbb{R})= U \oplus V\) 이면 \(\boldsymbol{A}\) 는 어떤 직교변환에 의해 블럭대각행렬이 된다.

(증명). \(\boldsymbol{u}\in U\) 이면 \(\boldsymbol{Au}\in U\) 이고 \(\boldsymbol{v}\in V\) 이면 \(\boldsymbol{Av}\in V\) 이다. \(\mathcal{M}_{n }(\mathbb{R}) = U \oplus V\) 이므로 \(U\) 의 정규직교기저 \(\{\boldsymbol{e}_i\}\) 와 \(V\) 의 정규직교기저 \(\{\boldsymbol{f}_j\}\) 에 대해 \(\{\boldsymbol{e}_i\}\cup \{\boldsymbol{f}_j\}\) 는 \(\mathcal{M}_{n }(\mathbb{R})\) 의 기저이며 이 기저로의 직교변환에 대해 \(\boldsymbol{A}\) 는 블럭대각행렬이 된다. \(\square\)


보조정리 7.6 \(\boldsymbol{A}\) 가 대칭행렬이면 어떤 직교 변환에 의해 대각행렬이 된다.

(증명). \(n\) 에 대한 수학적 귀납법을 이용한다. \(\boldsymbol{A}\in \mathcal{M}_{n \times n}(\mathbb{R})\) 이 대칭행렬이라고 하자. \(n=1\) 일 때는 스스로 대각행렬이다. 이제 \(k<n\) 인 대칭행렬은 어떤 직교 변환에 의해 대각행렬이 된다고 가정하자.

\(\boldsymbol{A}\) 가 \(n \times n\) 대칭 행렬일 때 고유값이 항상 존재하며 고유값에 대한 고유벡터 가운데 하나 \(\boldsymbol{u}_1\) 를 이용하여 \(U=\text{span}(\boldsymbol{u}_1)\) 를 만들면 \(U\) 은 불변부분공간이며 보조정리 7.4 에서 보였듯이 \(U^\perp\) 도 불변부분공간이다. \(\mathcal{M}_{n \times n}(\mathbb{R}) = U \otimes U^\perp\) 이므로 보조정리 7.5 에서 보였듯이 어떤 직교 변환에 의해 \(1\) 행렬과 \((n-1)\times (n-1)\) 행렬로 블록대각행렬이 된다. \((n-1)\times (n-1)\) 행렬은 귀납법의 가정에 의해 대각행렬이 되므로 \(\boldsymbol{A}\) 는 대각행렬이 된다. \(\square\)


정리 7.3 (실수체(\(\mathbb{R}\)) 에서 정의된 행렬의 스펙트럼 정리) \(\boldsymbol{A} \in \mathcal{M}_{n \times n}(\mathbb{R})\) 행렬에 대해 다음 세 명제는 동치이다.

 (\(1\)) \(\boldsymbol{A}\) 는 대칭행렬이다.

 (\(2\)) \(\boldsymbol{A}\) 는 직교변환에 의해 대각행렬로 표현된다.

 (\(3\)) \(\boldsymbol{A}\) 의 고유벡터로 \(\mathcal{M}_{n }(\mathbb{R})\) 의 정규직교기저를 구성 할 수 있다.

(증명). (\(1\implies 2\)) 는 앞에서 보였다.

(\(2 \implies 3\)) \(\boldsymbol{A}\) 가 orthognal 행렬 \(\boldsymbol{P}\) 에 의해 대각행렬로 변환된다고 하자. 즉 \(\boldsymbol{P}^{-1}\boldsymbol{AP}=\boldsymbol{D}\) 는 대각행렬이다. 대각행렬이 주어졌을 때, 표준정규직교벡터 \(\{\boldsymbol{e}_1,\ldots,\,\boldsymbol{e}_n\}\) 는 \(\lambda_i = D_{ii}\) 를 고유값으로 갖는 대각행렬의 고유벡터이다. 여기서

\[ \boldsymbol{APe}_i = \boldsymbol{P}\boldsymbol{P}^{-1}\boldsymbol{APe}_i= \boldsymbol{PDe}_i=\lambda_i \boldsymbol{Pe}_i \]

이므로 \(\boldsymbol{Pe}_i\) 는 \(\boldsymbol{A}\) 의 고유벡터이다. \(\boldsymbol{P}\) 가 linear isometry 이므로 \(\{\boldsymbol{Pe}_1,\ldots,\,\boldsymbol{Pe}_n\}\) 은 \(\boldsymbol{A}\) 의 고유벡터로 이루어진 \(\mathcal{M}_{n }(\mathbb{R})\) 의 정규직교기저이다.

(\(3 \implies 1\)) \(\{\boldsymbol{u}_1,\ldots,\,\boldsymbol{u}_n\}\) 이 \(\boldsymbol{A}\) 의 고유벡터로 이루어진 \(\mathbb{R}^n\) 의 정규직교기저라 하자. 이 때 \(\boldsymbol{u}_i\) 의 고유값은 \(\lambda_i\) 라 하자. \(\lambda_i\) 는 실수이므로,

\[ \langle \boldsymbol{A}^\ast \boldsymbol{u}_i,\,\boldsymbol{u}_j\rangle = \langle \boldsymbol{u}_i,\, \boldsymbol{Au}_j\rangle = \lambda_j \delta_{ij} = \lambda_i \delta_{ij} = \langle \boldsymbol{Au}_i,\, \boldsymbol{u}_j\rangle \]

이다. 즉 \(\boldsymbol{A}=\boldsymbol{A}^\ast\) 이며 \(\boldsymbol{A}\) 가 실행렬이므로 \(\boldsymbol{A}\) 는 대칭행렬이다. \(\square\)
