14  다중선형 형식

14.1 다중선형 형식과 쌍선형 형식

14.1.1 쌍선형 형식의 정의

정의 14.1 (\(k\)-다중선형 형식, 쌍선형 형식, \(k\)-텐서) \(V_1,\ldots,\,V_k\) 가 \(\mathbb{F}\)-벡터공간이라고 하자. 함수 \(T : V_1 \times \cdots \times V_k \to \mathbb{F}\) 가 각각의 \(V_i\) 에 대해 선형일 때 \(T\) 를 \(k\)-다중선형 형식(\(k\)-linear form) 이라고 한다. 즉 임의의 \(i\in \{1,\ldots,\,k\}\) 에 대해 \(v_i,\,v_i'\in V_i\), \(c\in \mathbb{F}\) 일 때,

\[ T (\cdots, v_i+cv_i', \cdots) = T(\cdots,\, v_i ,\cdots) + c T (\cdots,\, v_i',\,\cdots) \]

를 만족하면 \(T\) 는 \(k\)-linear form 이다. 특히 \(k=2\) 일 때 특별히 쌍선형 형식(bilinear form) 이라고 한다. 그리고 \(V_1=\cdots =V_k = V\) 이며 \(T\) 가 \(k\)-linear form 일 때 \(T\) 를 \(V\) 에 대한 \(k\)-linear form 혹은 \(V\) 에 대한 \(k\)-텐서 (\(k\)-tensor)라고 한다. \(V\) 에 대한 \(k\)-linear form 의 집합을 \(\mathcal{T}^{k}(V)\) 라고 쓴다.


보기 14.1 \(T\) 가 벡터공간 \(V\) 에 대한 \(k\)-다중선형형식 일 때 다음이 성립한다.

\[ T(\cdots, 0, \cdots ) = 0 \]


내적과 쌍선형 형식은 모두 \(U \times U \mapsto \mathbb{F}\) 함수이다. 그러나 쌍선형 형식은 내적과 두가지 면에서 다르다. 우선 \(\mathbb{C}\)-내적벡터공간에서 정의된 내적 \(\langle\,,\,\rangle\) 과 bilinear form \(T\) 의 경우 \(\langle u,\, cv\rangle = \overline{c}\langle u,\,v\rangle\) 이지만 \(T(u,\,cv)= cT(u,\,v)\) 이다. 또한 \(\mathbb{R}\)-내적벡터공간에서 \(\langle u,\,v\rangle = \langle v,\,u\rangle\) 이지만 쌍선형 형식은 일반적으로 \(T(u,\,v)\ne T(v,\,u)\) 이다.


정의 14.2 (다중선형 형식의 합과 스칼라곱) \(\mathbb{F}\)-벡터공간 \(V_1,\ldots,\,V_k\) 에서 정의된 \(k\)-다중선형형식 \(S,\,T\) 와 \(c\in \mathbb{F}\) 에 대해 합과 스칼라곱을 아래와 같이 정의한다.

\[ (S+aT)(v_1,\ldots,\,v_k) := S(v_1,\ldots,\,v_k) + c\cdot T(v_1,\ldots,\,v_k) \]


명제 14.1 \(\mathbb{F}\)-벡터공간 \(V_1,\ldots,\,V_k\) 에서 정의된 \(k\)-다중선형형식의 집합 \(\mathfrak{M}_k\) 는 정의 14.2 에 의해 벡터공간이다.


(증명). 우선 \(0_T: \mathfrak{M}_k \to \mathbb{F}\) 를 다음과 같이 정의한다.

\[ 0_M(v_1,\ldots,\,v_k) := 0 \tag{14.1}\]

이렇게 정의된 \(0_M\) 는 \(\mathfrak{M}_k\) 에 포함된다. \(T_1,\, T_2\in \mathfrak{M}_k,\, c\in \mathbb{F}\) 에 대해 합과 스칼라곱을 다음과 같이 정의한다.

\[ \begin{aligned} (T_1 + c T_2)(v_1,\ldots,\,v_k) := T_1 (v_1,\ldots,\,v_k) + cT_2 (v_1,\ldots,\,v_k) \end{aligned} \tag{14.2}\]

그렇다면 \(T_1 + cT_2\in \mathfrak{M}_k\) 이다. 이를 통해 우리는 \(\mathfrak{M}_k\) 가 벡터공간임을 알 수 있다. \(\square\)


여기서는 일반적인 \(k\)-다중형식보다는 쌍선형 형식과 \(k\)-텐서에 대해 집중해서 다루기로 한다. 보통 쌍선형 형식도 서로 다른 \(U,\,V\) 에 대해 정해지는 것보다는 2-텐서를(즉 \(U=V\) 인 경우) 더 많이 사용한다. 벡터공간 \(V\) 에 대한 쌍선형 형식이라고 할 때는 \(V\) 에 대한 \(2\)-텐서 를 의미한다. 이제 명제 14.1 로부터 자연스럽게 아래의 결론을 내릴 수 있다.


명제 14.2 벡터공간 \(V\) 에서 정의된 쌍선형 형식의 집합은 벡터공간이다. 벡터공간 \(V\) 에서 정의된 \(k\)-텐선의 집합 \(\mathcal{T}^k(V)\) 는 벡터공간이다.


14.1.2 쌍선형 형식의 행렬 표현

정의 14.3 (쌍선형 형식의 행렬표현) 유한차원 \(\mathbb{F}\)-벡터공간 \(V\) 에서의 쌍선형 형식 \(T\in \mathcal{T}^2(V)\) 을 생각하자. \(\mathcal{B}=(v_1,\ldots,\,v_n)\) 이 \(V\) 의 기저라고 할 때 \(T\) 의 \(\mathcal{B}\) 에 대한 행렬표현 \([T]_{\mathcal{B}}\) 를 다음과 같이 정의한다.

\[ ([T]_{\mathcal{B}})_{i, j} = T(v_i, v_j). \]


명제 14.3 정의 14.3 에서 정의된 \([T]_{\mathcal{B}}\) 에 대해 다음이 성립한다.

\[ T(v, v') = \left([v]_\mathcal{B}\right)^T [T]_\mathcal{B} [v']_\mathcal{B}. \]


(증명). \(v=\sum_i a_i v_i,\, v'=\sum_j b_j v_j\) 라 하자.

\[ \begin{aligned} T(v, v') &= \sum_{i, j} T(a_i v_i, b_j, v_j) = \sum_{i, j} a_i b_j T(v_i, v_j) = \sum_{i, j} ([v]_{\mathcal{B}})_i ([T]_{\mathcal{B}})_{ij} ([v']_{\mathcal{B}})_j\\ &= \left([v]_\mathcal{B}\right)^T [T]_\mathcal{B} [v']_\mathcal{B}. \qquad \square \end{aligned} \]


명제 14.4 \(\boldsymbol{A}\in \mathcal{M}_{n \times n}(\mathbb{F})\) 에 대해 아래와 같이 정의된 \(B:\mathcal{M}_n(\mathbb{F}) \times \mathcal{M}_n(\mathbb{F}) \to \mathbb{F}\) 는 \(\mathcal{M}_n(\mathbb{F})\) 에 대한 쌍선형 형식이다.

\[ B(\boldsymbol{u}, \boldsymbol{v}) = \boldsymbol{u}^T\boldsymbol{A}\boldsymbol{v}. \]


(증명). trivial.


위의 두 명제를 가만히 생각해보자. 쌍선형 형식을 행렬로 생각 할 수 있고, 임의의 행렬을 이용하여 쌍선형 형식을 정의 할 수 있다. 행렬과 선형사상의 관계를 밝힐 때 했던 것을 여기서 다시 한 번 해보자.


정리 14.1 \(n\) 차원 \(\mathbb{F}\)-벡터공간 \(V\) 의 기저 \(\mathcal{B}_V = (v_1,\ldots,\,v_n)\) 을 생각하자. 아래에 정의된 \(\Phi_\mathcal{B} : \mathcal{T}^2(V) \to \mathcal{M}_{n\times n}(\mathbb{F})\) 는 동형사상이다.

\[ \Phi_\mathcal{B} (T) = [T]_\mathcal{B}. \]


(증명). \(\Phi_\mathcal{B}\) 가 선형사상임을 보이자. \(S,\,T\in \mathcal{T}^2(V),\, a\in \mathbb{F}\) 일 때

\[ ([S+aT]_\mathcal{B})_{ij} = (S+aT)(v_i, v_j) = S(v_i, v_j) + aT(v_i, v_j) = ([S]_{\mathcal{B}})_{ij} + a\cdot ([T]_{\mathcal{B}})_{ij} \]

이다. 따라서 \(\Phi_\mathcal{B}(S+aT) = \Phi_\mathcal{B}(S) + a \Phi_\mathcal{B}(T)\) 이므로 \(\Phi_B\) 는 선형사상이다.

임의의 \(\boldsymbol{A}\in \mathcal{M}_{n \times n}(\mathbb{F})\) 와 \(v_1,\,v_2 \in V\) 에 대해 \(([v_1]_\mathcal{B})^T \boldsymbol{A} [v_2]_\mathcal{B}\) 는 \(V\) 에서의 쌍선형 형식이다. 따라서 \(\Phi_\mathcal{B}\) 는 전사이다. 또한 명제 14.3 에 따라 \([T]_\mathcal{B}\) 는 유일하게 정해진다. 따라서 \(\Phi_\mathcal{B}\) 는 단사이다. 즉 \(\Phi_\mathcal{B}\) 는 동형사상이다.


14.2 대칭형식과 교대형식

14.2.1 대칭형식 과 교대형식의 정의

정의 14.4 (대칭형식 교대형식) \(T\in \mathcal{T}^k(V)\) 에 대해 \(T\) 에 입력되는 두 벡터의 위치를 바꾸어도 그 값이 바뀌지 않는다면 \(T\) 를 \(V\) 에 대핸 \(k\)-대칭형식(\(k\)-symmetric form) 이라고 한다. 즉 \(T\) 가 \(k\)-대칭형식이면 임의의 \(v,\,v'\in V\) 에 대해 다음이 성립한다.

\[ T(\cdots, v, \cdots,v', \cdots) = T(\cdots, v',\cdots, v,\cdots) \tag{14.3}\]

반대로 \(T\in \mathcal{T}^k(V)\) 에 대해 \(T\) 입력되는 두 벡터의 위치를 바꾸었을 때 그 값이 항상 \(0\) 이 되면 \(T\) 를 \(V\) 에 대한 \(k\)-교대형식(\(k\)-alternating form) 이라고 한다. 즉 임의의 \(v\in V\) 에 대해

\[ T(\cdots, v, \cdots,v, \cdots) =0 \tag{14.4}\]

이면 \(T\) 는 \(V\) 에 대한 \(k\)-교대형식 이다. \(V\) 에 대한 \(k\)-대칭형식의 집합을 \(\mathcal{S}^k(V)\) 라고 쓰고 \(k\)-교대형식 의 집합을 \(\mathcal{A}^k(V)\) 라고 쓴다.


명제 14.5 \(T\in \mathcal{S}^k(V)\) 에 대해 다음은 동치임을 보여라.

 (\(1\)) \(T\) 는 \(k\)-교대형식이다. 즉 \(T\in \mathcal{A}^k(V)\) 이다.

 (\(2\)) \(T\) 에 입력되는 벡터들 가운데 두개의 위치를 바꾸면 결과값은 원래값에 \(-1\) 을 곱한값이다. 즉

\[ T( \cdots, v_i, \cdots, v_j ,\cdots) = - T(\cdots, v_j, \cdots, v_i, \cdots) \tag{14.5}\]

이다.


(증명). (\(1 \implies 2\)) 임의의 두 벡터의 위치를 선택하여 \(T(\cdots, v, \cdots, v', \cdots)\) 를 \(T(v,\,v)\) 라고 표기하자. 교대형식의 정의에 의해 \(T(v,v)=0\) 이다. \[ \begin{aligned} 0 = T(v_i + v_j, v_i + v_j) &= T(v_i, v_i) + T(v_i, v_j) + T(v_j, v_i) + T(v_j, v_j) = T(v_i, v_j)+ T(v_j, v_i) \end{aligned} \]

이므로 \(T(v_i, v_j)=-T(v_j, v_i)\) 이다.

(\(2 \implies 1\)) \(T(v,v)= -T(v,v)\) 이므로 \(T(v,v)=0\) 이다. \(\square\)


즉 교대형식의 정의 식 14.4 과 식 14.5 는 동치이다.


우리는 앞서 \(\mathcal{T}^k(V)\) 가 벡터공간임을 보였다. \(\mathcal{S}^k(V)\) 와 \(\mathcal{A}^k(V)\) 는 \(\mathcal{T}^k(V)\) 의 부분집합이기 때문에 당연한 질문에 답해야 한다.


명제 14.6 \(\mathcal{S}^k(V)\) 와 \(\mathcal{A}^k(V)\) 는 \(\mathcal{T}^k(V)\) 의 부분공간이다.


(증명). 식 14.1 의 \(0_M\) 는

\[ 0_M(\cdots, v_i, \cdots,v_j, \cdots) = 0_M(\cdots, v_j,\cdots, v_i,\cdots)=0 \]

이므로 \(0_M\in \mathcal{S}^k(V)\) 이다. \(S_1,\,S_2\in \mathcal{S}^k(V),\, c\in \mathbb{F}\) 에 대해 \(S_1 + cS_2\in \mathcal{T}^k(V)\) 임을 쉽게 보일 수 있다. 따라서 \(\mathcal{S}^k(V)\le \mathcal{T}^k(V)\) 이다. 같은 방법으로 \(\mathcal{A}^k(V) \le \mathcal{T}^k(V)\) 임을 보일 수 있다. \(\square\)


\(k\)-교대형식을 다시 보자. 우리는 이것을 이미 행렬식 에서 보았다. 행렬식은 \(n\) 차원 벡터 \(V\) 에 대한 \(n\)-교대형식의 대표적인 예이다. 이제부터 우리는 \(\mathcal{S}^2(V)\) 와 \(\mathcal{A}^2(V)\) 에 집중하기로 하자. 우리는 쌍선형 형식이 행렬로 표현될 수 있다는 것을 안다. 당연히 대칭형식과 교대형식도 행렬로 표현될 수 있는데, 각각의 행렬은 대칭적인 특성을 가지고 있다. 우선 미리 행렬의 형식에 대해 정의하자.


정의 14.5 (대칭행렬, 반대칭 행렬) \(\boldsymbol{A}\in \mathcal{M}_{n\times n}(\mathbb{F})\) 가 모든 \(1\le i,\,j\ne n\) 에 대해 \(A_{ij}= A_{ji}\) 일 때 \(\boldsymbol{A}\) 를 대칭행렬(symmectric matrix) 라고 한다. 모든 \(1 \le i \le j \le n\) 에 대해 \(A_{ij} = -A_{ji}\) 일 때 \(\boldsymbol{A}\) 를 교대행렬(alternaing matrix) 혹은 반대칭 행렬(skew-symmetric matrix) 이라고 한다.


보기 14.2 \(\boldsymbol{A}\) 가 교대행렬이라면 \(A_{ii} = - A_{ii}\) 이므로 모든 대각성분은 \(0\) 이다.


이제 대칭형식과 교대형식에 대한 행렬표현에 대해 알아보자.


명제 14.7 유한차원 벡터공간 \(V\) 와 그 기저 \(\mathcal{B}=(v_1,\ldots,\,v_n)\) 에 대해 아래 두 명제는 동치이다.

 (\(1\)) \(T\in \mathcal{S}^2(V)\) 이다.

 (\(2\)) \([T]_\mathcal{B}\) 는 대칭행렬이다.


(증명). \(T(v_i, v_j)= ([T]_\mathcal{B})_{ij}\), \(T(v_j, v_i)= ([T]_{\mathcal{B}})_{ji}\) 이다. 따라서, \[ T\in \mathcal{S}^2(V) \iff \forall v_1,\,v_2\in V,\, T(v_1, v_2)= T(v_2, v_1) \iff ([T]_\mathcal{B})_{ij} = ([T]_{\mathcal{B}})_{ji} \]

이다. \(\square\)


명제 14.8 유한차원 벡터공간 \(V\) 와 그 기저 \(\mathcal{B}=(v_1,\ldots,\,v_n)\) 에 대해 아래 두 명제는 동치이다.

 (\(1\)) \(T\in \mathcal{A}^2(V)\) 이다.

 (\(2\)) \([T]_\mathcal{B}\) 는 교대행렬이다.


(증명). \(T(v_i, v_j)= ([T]_\mathcal{B})_{ij}\), \(T(v_j, v_i) = ([T]_{\mathcal{B}})_{ji}\) 이다. 따라서, \[ T\in \mathcal{A}^2(V) \iff \forall v_1,\,v_2\in V,\, T(v_1, v_2)= -T(v_2, v_1) \iff ([T]_\mathcal{B})_{ij} = -([T]_{\mathcal{B}})_{ji} \]

이다. \(\square\)


이제 벡터공간으로서의 \(\mathcal{T}^2(V)\) 와 그 부분공간으로서의 \(\mathcal{S}^2(V),\, \mathcal{A}^2(V)\) 의 성질에 대해 알아보자.


명제 14.9 유한차원 \(\mathbb{F}\)-벡터공간 \(V\) 에 대해 다음이 성립한다.

  (\(1\)) \(\mathcal{S}^2(V)\) 과 \(\mathcal{A}^2(V)\) 는 \(\mathcal{T}^2(V)\) 의 부분공간이다.

  (\(2\)) \(\dim(V)=n\) 일 때 \(\dim(\mathcal{S}^2(V)) = \dfrac{n(n+1)}{2}\) 이며 \(\dim(\mathcal{A}^2(V)) = \dfrac{n(n-1)}{2}\) 이다.

  (\(3\)) \(\mathcal{T}^2(V) = \mathcal{S}^2(V) \oplus \mathcal{A}^2(V)\) 이다.


(증명). \(V\) 의 기저 \(\mathcal{B} = (v_1,\ldots,\,v_n)\) 을 생각하자.

(\(1\)) 명제 14.6

(\(2\)) \(\boldsymbol{E}^{(i, j)} \in \mathcal{M}_{n\times n}(\mathbb{F})\) 를 \(i\) 행 \(j\) 열 성분은 \(1\), 나머지 \(n^2-1\) 개의 성분은 \(0\) 인 행렬이라고 하자. \(\{\boldsymbol{E}^{(i, j)}: 1 \le i,\,j\le n\}\) 은 \(\mathcal{M}_{n \times n}(\mathbb{F})\) 의 기저임은 쉽게 알 수 있다. 이 때,

\[ \{\boldsymbol{E}^{(i, i)}: 1 \le i \le n \} \cup \{ \boldsymbol{E}^{(i, j)}+ \boldsymbol{E}^{(j, i)} : 1 \le i < j \le n\} \]

은 선형 독립이며 \(n\times n\) 대칭행렬을 span 한다는 것을 알 수 있다. \(n \times n\) 대칭행렬의 집합과 \(S^2(V)\) 사이에 동형사상이 존재하므로 \(\dim(\mathcal{S}^2(V)) = \dfrac{n(n+1)}{2}\) 이다. 또한,

\[ \{ \boldsymbol{E}^{(i, j)}- \boldsymbol{E}^{(j, i)} : 1 \le i < j \le n\} \]

는 선형독립이며 \(n\times n\) alternating 행렬을 span 한다. 따라서 \(\dim(\mathcal{A}^2(V)) = \dfrac{n(n-1)}{2}\) 이다.

(\(3\)) \(\boldsymbol{A} = \dfrac{1}{2}(\boldsymbol{A}+\boldsymbol{A}^T) + \dfrac{1}{2} (\boldsymbol{A}-\boldsymbol{A}^T)\) 이다. 여기서 \(\dfrac{1}{2}(\boldsymbol{A}+\boldsymbol{A}^T)\) 는 대칭행렬이며 \(\dfrac{1}{2}(\boldsymbol{A}-\boldsymbol{A}^T)\) 는 alternating 행렬이므로 \(\mathcal{T}^2(V) = \mathcal{S}^2(V) + \mathcal{A}^2(V)\) 이다. 또한 \(\mathcal{S}^2(V) \cap \mathcal{A}^2(V)=\{0\}\) 이므로 \(\mathcal{T}^2(V) = \mathcal{S}^2(V) \oplus \mathcal{A}^2(V)\) 이다. \(\square\)


14.2.2 교대형식과 행렬식

우리가 대칭군을 이용한 행렬식 에서 행렬식을 도입할 때 우선 행렬의 행벡터에 대한 \(\det\) 함수로 도입하였다. 행벡터로 도입한것은 우리가 바로 전에 행 기본연산에 대해 논의했기 때문에 그 연속선상에서 그렇게 했다. 우리는 이미 열벡터에 대해서도 정의할 수 있다는 것을 안다. \(\boldsymbol{A}_i\) 를 \(\boldsymbol{A}\) 의 \(i\) 번째 열벡터라고 하자.

\[ \phi (\boldsymbol{A}_{1},\ldots,\,\boldsymbol{A}_{k}) := \det (\boldsymbol{A}) \]

라고 하하면 \(\phi\) 함수는 다음과 같은 성질을 가진다는 것을 안다.

 (\(1\)) \(\phi (\boldsymbol{e}_1,\ldots,\,\boldsymbol{e}_k)=1\)

 (\(2\)) \(\phi \in \mathcal{A}^k(\mathcal{M}_k(\mathbb{F}))\)

이제 우리는 반대로 위의 두 조건을 만족하는 \(k\)-alternating form 은 유일하다는 것을 보일 것이다.


보조정리 14.1 \(\mathcal{M}_k(\mathbb{F})\) 에 대해 다음이 성립한다.

\[ \dim (\mathcal{A}^k (\mathcal{M}_k(\mathbb{F})))=1. \]


(증명). 우선 \(\dim (\mathcal{A}^k (\mathcal{M}_k(\mathbb{F})))\ne 0\) 임을 보이자. 대칭군을 사용하여,

\[ \phi(\boldsymbol{A}_1,\ldots,\,\boldsymbol{A}_k) := \sum_{\sigma \in S_k} \text{sgn}(\sigma) \prod_{i=1}^k A_{\sigma(i),i} \tag{14.6}\]

라고 정의하자. 우리는 \(\phi \in \mathcal{T}^k(\mathcal{M}_k(\mathbb{F}))\) 임을 쉽게 보일 수 있으며 연습문제 3.3 에서 보인대로 \(\phi \in \mathcal{A}^k (\mathcal{M}_k(\mathbb{F}))\) 이다. 이 때 \(\phi (\boldsymbol{e}_1,\ldots,\,\boldsymbol{e}_k)=1\) 이므로 \(\dim (\mathcal{A}^k (\mathcal{M}_k(\mathbb{F}))) > 0\) 이다.

이제 \(\psi \in \mathcal{A}^k(\mathcal{M}_k(\mathbb{F}))\) 이며 \(\psi (\boldsymbol{e}_1,\ldots,\,\boldsymbol{e}_k) = c\) 라고 하자. 여기에 대해 \(\chi := c\phi -\psi\) 라고 정의하면 \(\chi \in \mathcal{A}^k(\mathcal{M}_k(\mathbb{F}))\) 이며 \(\chi(\boldsymbol{e}_1,\ldots,\boldsymbol{e}_k)=0\) 이다.

그런데 모든 \(\boldsymbol{v}\in \mathcal{M}_k(\mathbb{F})\) 는 \(\{\boldsymbol{e}_1,\ldots,\,\boldsymbol{e}_k\}\) 의 선형결합이다. \(i=1,\ldots,\,k\) 에 대해 \(\boldsymbol{v}_i = \sum_j A_{ij}\boldsymbol{e}_j\) 라고 하자.

\[ \begin{aligned} \chi(\boldsymbol{v}_1,\ldots,\,\boldsymbol{v}_k) &= \sum_{j_1=1}^k A_{1, j_1}\chi (\boldsymbol{e}_{j_1},\boldsymbol{v}_2,\ldots,\,\boldsymbol{v}_k) \\ & \qquad \vdots \\ &= \sum_{j_1=1}^k \cdots \sum_{j_k=1}^k A_{1, j_1}\cdots A_{k,\,j_k} \chi(\boldsymbol{e}_{j_1},\ldots,\,\boldsymbol{e}_{j_k}) \end{aligned} \]

이다. 그런데 \(\chi\) 안의 표준기저의 인덱스 가운데 어떤 두개가 같으면 alternating \(k\) form 의 정의에 의해 \(\chi\) 함수값이 \(0\) 이다. 모두 다르면 \(\chi\) 함수의 정의에 의해 \(0\) 이다. 따라서 임의의 \(\boldsymbol{v}_1,\ldots,\,\boldsymbol{v}_n\) 에 대해 \(\chi(\boldsymbol{v}_1,\ldots,\,\boldsymbol{v}_n)=0\) 이다.

이것은 \(\mathcal{A}^k(\mathcal{M}_k(\mathbb{F}))\) 의 모든 원소는 식 14.6 에서 정의된 \(\phi\) 의 상수배라는 의미이다. 즉 \(\dim (\mathcal{A}^k (\mathcal{M}_k(\mathbb{F}))) = 1\) 이다. \(\square\)


이로부터 자연스럽게 행렬식의 유일성이 증명된다.


정리 14.2 (행렬식의 유일성) \(\phi \in \mathcal{A}^k(\mathcal{M}_k(\mathbb{F}))\) 이며 \(\phi (\boldsymbol{e}_1,\ldots,\,\boldsymbol{e}_k)=1\) 인 \(\phi\) 는 유일하다.


14.3 이차형식

14.3.1 이차형식

정의 14.6 (이차형식) 벡터공간 \(V\) 에서 정의된 함수 \(\Psi : V\to \mathbb{F}\) 가 임의의 \(c\in \mathbb{F},\, v,\,v'\in V\) 에 대해

  (\(1\)) \(B_\Psi (v, v') = \Psi (v+v') - \Psi (v) - \Psi (v')\) 이 \(V\) 에 대한 대칭인 쌍선형 형식 이며,

  (\(2\)) \(\Psi (cv) = c^2 \Psi (v)\)

를 만족 할 때 \(\Psi\) 를 \(V\) 에 대한 이차형식(quandratic form) 이라고 한다.


명제 14.10 \(T\in \mathcal{S}^2(V)\) 일 때 \(\Psi_T(v) = \frac{1}{2}T(v, v)\) 는 이차형식이다.


(증명). 우선 조건 (\(1\)) 을 만족한다는 것을 보이자. \[ \begin{aligned} B_\Psi(v, v') &= \Psi_T(v+v') - \Psi(v)-\Psi(v') \\ &= \dfrac{1}{2}T(v+v', v+v') - \dfrac{1}{2}T(v, v)- \dfrac{1}{2}T(v', v') = T(v, v') \end{aligned} \]

이므로 \(B_\Psi \in \mathcal{S}^2(V)\) 이다. 또한

\[ \Psi_T(cv) = \dfrac{1}{2}T(cv, cv) = \dfrac{1}{2}cT(v, cv)= \dfrac{1}{2}c^2T(v) = c^2 \Psi_T(v) \]

이므로 (\(2\)) 를 만족한다. 따라서 \(\Psi_T\) 는 \(V\) 에 대한 이차형식이다. \(\square\)


명제 14.10 로부터 우리는 대칭 쌍선형형식 대한 이차형식이 존재함을 안다. 이제 좀 더 깊이 알아보자.


명제 14.11 \(\mathbb{F}\)-벡터공간 \(V\) 에서의 이차형식 전체의 집합을 \(\mathcal{Q}_V\) 라고 하자. \(0_Q : V \to \mathbb{F}\) 를 다음과 같이 정의한다.

\[ \forall v\in V,\, 0_Q(v) = 0 \]

\(\Psi, \Phi\in Q_V,\, a\in \mathbb{F}\) 에 대해

\[ (\Psi + c \Phi)(v) = \Psi (v) + a\Phi(v) \]

라고 하자. 이 때 \(\mathcal{Q}_V\) 는 \(\mathbb{F}\)-벡터공간이다.


(증명). \(0_Q \in \mathcal{Q}_V\) 임은 쉽게 보일 수 있다. 또한 임의의 \(\Psi\in \mathcal{Q}_V\) 에 대해 \(0_Q + \Psi = \Psi+0_Q =\Psi\) 라는 것도 쉽게 보일 수 있다. \[ \begin{aligned} B_{\Psi + a\Phi} (v, v') &= (\Psi+a\Phi)(v+v') - (\Psi + a\Phi)(v) - (\Psi + a\Phi)(v') \\ &= B_\Psi(v, v') + aB_\Phi (v, v') \end{aligned} \]

이며 \(B_\Psi,\, B_\Phi \in \mathcal{S}^2(V)\) 이므로 \(B_\Psi + aB\Phi\in \mathcal{S}^2(V)\) 이다. 또한,

\[ (\Psi + a\Phi)(cv) = \Psi (cv) + a\Phi (cv) = c^2 (\Psi + a\Phi)(v) \]

이므로 \(\Psi + a\Phi \in \mathcal{Q}_V\) 이다. 이로부터 \(\mathcal{Q}_V\) 가 벡터공간이라는 것을 벡터공간의 정의에 맞추어 증명 할 수 있다. \(\square\)


정리 14.3 벡터공간 \(V\) 에 대해 \(\mathcal{S}^2(V)\) 와 \(\mathcal{Q}_V\) 는 동형이다.


(증명). \(\Gamma : \mathcal{Q}_V \to \mathcal{S}^2(V)\) 를 정의 14.6 에 따라 \(\Gamma (\Psi) = B_\Psi\) 로 정의하자. 명제 14.10 에서 \(T\in \mathcal{S}^2(V)\) 일 때 \(\Psi_T(v) = \dfrac{1}{2}T(v, v)\) 는 quadratic form 이며

\[ \begin{aligned} B_{\Psi_T}(v, v) = \Gamma (\Psi_T) (v, v') &= \Psi_T(v+v') - \Psi_T(v) - \Psi_T(v') \\ &= \dfrac{1}{2}T(v+v', v+v') - \dfrac{1}{2}T(v, v) - \dfrac{1}{2}T(v', v') \\ &= T(v, v') \end{aligned} \]

이다. 즉 \(\Gamma\) 는 전사이다. 또한 \(\Psi,\, \Phi \in \mathcal{Q}_V\) 에 대해 \(B_\Psi = B_\Phi\) 이면 \(\Phi = \Psi\) 이므로 \(\Gamma\) 는 단사이다.

이제 \(\Gamma\) 가 선형사상임을 보이면 된다. 즉 \(\Phi,\, \Psi\in \mathcal{Q}_V,\, c\in \mathbb{F}\) 에 대해 \(\Gamma (\Phi + c\Psi) = \Gamma (\Phi) + c\Gamma(\Psi)\) 임을 보이면 된다.

\[ \begin{aligned} \Gamma (\Phi + c\Psi)(v, v') & = (\Phi + c\Psi)(v+v') - (\Phi+c\Psi)(v)- (\Phi + c\Psi)(v') \\ &= \Phi(v+v') - \Phi(v) - \Phi (v') + c\Psi(v+v') - c\Psi(v) - c\Psi(v') \\ &= (\Gamma (\Phi) + c\Gamma (\Psi))(v, v'). \end{aligned} \]

따라서 \(\mathcal{Q}_V \cong \mathcal{S}^2(V)\) 이다. \(\square\)


14.3.2 이차형식의 행렬 표현

이제 \(n\) 차원 \(\mathbb{F}\)-벡터공간 \(V\) 에 대해 \(\Psi \in \mathcal{Q}_V\) 라고 하자. 우리는 \(B_\Psi(v, v) = \dfrac{1}{2} \Psi(v) \in \mathcal{S}^2(V)\) 라는 것을 알며, 따라서 \(\Psi_B\) 가 대칭행렬로 표현된다는 것도 안다. 이제 \(V\) 의 기저 \(\mathcal{B}=(v_1,\ldots,\,v_n)\) 을 생각하자. 명제 14.3 에 따라

\[ \Psi(v) = \dfrac{1}{2}B_\psi(v, v) = \dfrac{1}{2}([v]_\mathcal{B})^T[B_{\Psi}]_\mathcal{B} ([v]_\mathcal{B}) \]

이다.

\(x\in V\) 에 대해 \([x]_\mathcal{B} = \boldsymbol{x}=\begin{bmatrix}x_1 & \cdots &x_n\end{bmatrix}^T,\, [B_\Psi]_\mathcal{B}=\boldsymbol{A}\) 라고 하자. 그렇다면,

\[ \Psi(x) = \dfrac{1}{2}\boldsymbol{x}^T\boldsymbol{Ax} = \dfrac{1}{2} \sum_{i=1}^n A_{ii}{x_i}^2 + \sum_{i<j} A_{ij}x_i x_j \]

이다. 즉 이차형식은 동차인 2차식이 된다. 역으로

\[ f(x_1,\ldots,\,x_n) = \sum_{i=1} b_i {x_i}^2 + \sum_{i<j} c_{ij}x_i x_j \]

인 동차인 2차식은 \(A_{ii} = 2 b_i,\, A_{ij} = c_{ij}\) 에 대해 이차형식 이 된다는 것을 알 수 있다.


14.4 직교군과 심플렉틱 군

14.4.1 이차공간과 직교군

우리는 앞서 벡터공간 \(V\) 에 내적이 부여되면 내적벡터공간이라고 불렀다. 마찬가지로 어떤 이차형식이 부여되었을 경우에도 이름을 붙일 수 있다.

정의 14.7 (이차공간) 유한차원 \(\mathbb{F}\)-벡터공간 \(V\) 에서 어떤 이차형식 \(\Psi \in \mathcal{Q}_V\) 이 정해졌다면 이에 따라 대칭 쌍선형 형식 \(B_\Psi\) 도 정의된다. 이렇게 벡터공간에 이차형식 \(\Psi\) 이 부여되었을 때, 혹은 이와 동등하게 대칭 쌍선형 형식이 부여되었을 때 이 벡터공간 \(V\) 를 이차공간(quadratic space) 라고 하고 \((V,\, \Psi)\), \((V,\, B_\Psi,\, \Psi)\), 혹은 \((V, B_\Psi)\) 라고 쓴다

정의 14.8 (직교군) Quadratic space \((V, B_\Psi)\) 에 대해 다음과 같이 정의된 \(O(V, B_\Psi)\) 를 \((V, B_\Psi)\) 에 대한 직교군 (orthonormal group) 이라 한다.

\[ O(V,\, B_\Psi) = \{ L \in GL(V) : B_\Psi(Lv, Lv')=B_\Psi(v, v'),\,\forall v,\,v'\in V\}. \]

\(SL(V)\) 를 정의할 때와 마찬가지로 \(SO(V,\, \mathcal{B}_\Psi)\) 를 정의 할 수 있다.

\[ SO(V,\, B_\Psi) = \{ L\in O(V,\, B_\Psi) : \det(L) = 1\}. \]


직교군이라는 이름이 붙었다는 것은 \(O(V,\, B_\Psi)\) 가 군이라는 뜻이며 따라서 \(GL(V)\) 의 부분군일 것이라는 것을 예상 할 수 있다.


명제 14.12 유한차원 벡터공간 \(V\) 에 대해 다음이 성립한다.

 (\(1\)) \(O(V, B_\Psi) \le GL(V)\).

 (\(2\)) \(SO(V,\, B_\Psi) \trianglelefteq O(V,\, B_\Psi)\).


(\(1\)) \(V\) 에서의 항등사상 \(I_V\) 가 \(I_V\in O(V, \mathcal{B}_\Phi)\) 임은 자명하다. \(L_1,\,L_2\in O(V,\,B_\Psi)\) 에 대해 \(B(L_1L_2v,\, L_1L_2v')=B(v, v')\) 이므로 \(L_1L_2 \in O(V,\, B_\Psi)\) 이다. \(L\in O(V,\,B_\Psi)\) 이면,

\[ B_\Psi (L^{-1}v,\,L^{-1}v') = B_\Psi (LL^{-1}v, LL^{-1}v') = B_\Psi (v, v') \]

이미로 \(L^{-1} \in O(V,\, B_\Psi)\) 이다. 따라서 \(O(V,\, B_\Psi) \le GL(V)\) 이다.

(\(2\)) \(SO(V, B_\Psi) \le O(V,B_\Psi)\) 임은 쉽게 보일 수 있다. \(L\in O(V, B_\Psi),\, S\in SO(V, B_\Psi)\) 에 대해 \(\det(LSL^{-1})=1\) 이므로 \(SO(V,\, B_\Psi) \trianglelefteq O(V,\, B_\Psi)\) 이다. \(\square\)


연습문제 14.1 유한차원 벡터공간 \(V\) 와 그 기저 \(\mathcal{B}=(v_1,\ldots,\,v_n)\) 에 대해 다음이 성립한다.

\[ \begin{aligned} O(V,\,B_\Psi) &= \{L \in GL(V) : \Psi (Lv) = \Psi (v)\} \\ &= \{L\in GL(V) : B_\Psi(Lv_i, Lv_j) = B_\Psi (v_i, v_j),\,\forall v_i,\,v_j\in \mathcal{B} \} \end{aligned} \]

이다.


(해답). \(A,\,B\) 를 다음과 같이 정하자. \[ \begin{aligned} O_1 &=\{L \in GL(V) : \Psi (Lv) = \Psi (v)\}, \\ O_2 &=\{L\in GL(V) : B_\Psi(Lv_i, Lv_j) = B_\Psi (v_i, v_j),\,\forall v_i,\,v_j\in \mathcal{B} \}. \end{aligned} \]

우선 \(O(V, B_\Psi) = O_1\) 임을 보이자. \(L\in O(V, B_\Psi)\) 이면 \(\Psi (Lv) = B_\Psi (Lv, Lv) = B_\Psi(v, v) = \Psi (v)\) 이므로 \(L \in O_1\) 이다. 즉 \(O(V, B_\Psi) \subset O_1\) 이다. \(L\in O_1\) 이면 \(\Psi(Lv) = \Psi (v)\) 이고,

\[ B_\Psi (Lv, Lv') = \Psi (Lv+Lv') - \Psi (Lv)- \Psi(Lv') = \Psi (v+v')-\Psi(v)- \Psi (v') = B_\Psi (Lv) \]

이므로 \(L\in O(V, B_\Psi)\) 이다. 따라서 \(O(V,\,B_\Psi) = O_1\) 이다.

이제 \(O(V, B_\Psi) = O_2\) 임을 보이자. 두 집합의 정의에 의해 \(O(L, B_\Psi) \subset O_2\) 이다. 이제 \(L\in O_2\) 라고 하자. \(v=\sum_i a_i v_i,\, v' = \sum_j b_j v_j\) 라고 하면,

\[ B_\Psi (Lv, Lv') = \sum_{i, j} a_i b_j B_\Psi (Lv_i, Lv_j) = \sum_{i,j}B_\Psi (v_i, v_j) = B_\Psi (v, v') \]

이다. 따라서 \(O(L, B_\Psi) = O_2\) 이다. \(\square\)


명제 14.13 유한차원 벡터공간 \(V\) 와 quadratic form \(\Psi\) 에 대한 quadratic space \((V, \Psi)\) 를 생각하자. \(V\) 의 기저 \(\mathcal{B}\) 에 대해 다음은 동치이다.

  (\(1\)) \(L \in O(V, B_\Psi)\).

  (\(2\)) \(([L]_\mathcal{B})^T[B_\Psi]_\mathcal{B}([L]_\mathcal{B}) =[B_\Psi]_\mathcal{B}\).

(증명). \(v,\,v'\in V\) 에 대해 \([v]_\mathcal{B}=\boldsymbol{v},\, [v']_\mathcal{B}=\boldsymbol{v}'\) 이라 하고 \([L]_\mathcal{B}=\boldsymbol{L},\, [B_\Psi]_\mathcal{B}= \boldsymbol{B}\) 라고 하자. 우선 \(\Psi,\,\Psi\in \mathcal{Q}_V\) 일 때 \(B_\Psi (v, v) = B_\Phi (v, v) \iff \Psi = \Phi\) 임은 당연하다. 따라서 \(\boldsymbol{v}^T[B_\Psi]\boldsymbol{v} = \boldsymbol{v}^T[B_\Phi]\boldsymbol{v} \iff [B_\Psi]=[B_\Phi]\) 를 의미한다.

(\(1 \implies 2\)) 명제 14.3 에 따라 \(B_{\Psi}(Lv, Lv') = \boldsymbol{v}^T\boldsymbol{L}^T\boldsymbol{BLv}\), \(B_\Psi (v,v) = \boldsymbol{v}^T\boldsymbol{Bv}\) 이다. 따라서 \(\boldsymbol{L}^T\boldsymbol{BL}= \boldsymbol{B}\) 이다.

(\(2 \implies 1\)) \(\boldsymbol{L}^T\boldsymbol{BL}= \boldsymbol{B}\) 이면 \(\boldsymbol{v}^T\boldsymbol{L}^T\boldsymbol{BLv}\), \(B_\Psi (v,v) = \boldsymbol{v}^T\boldsymbol{Bv}\) 이므로 \(L\in O(V,B_\Psi)\) 이다. \(\square\)


정의 14.9 (대칭행렬에 대한 직교군) \(\boldsymbol{J}\in \mathcal{M}_{n \times n}(\mathbb{F})\) 가 대칭행렬 일 때

\[ O_n^\boldsymbol{J} (\mathbb{F}) = \{ \boldsymbol{A} \in GL_n(\mathbb{F}) : \boldsymbol{A}^T\boldsymbol{JA} = \boldsymbol{J}\} \]

를 \(\boldsymbol{J}\) 에 대한 대칭군이라고 한다.


보기 14.3 대칭행렬 \(\boldsymbol{J}\in \mathcal{M}_{n \times n}(\mathbb{F})\) 가 가역일 때 \(\boldsymbol{A}\in O_n^\boldsymbol{J}(\mathbb{F})\) 의 행렬식은 \(\det(\boldsymbol{A}^T\boldsymbol{JA}) = \boldsymbol{J}\) 이며 \(\det (\boldsymbol{A}^T) = \det (\boldsymbol{A})\) 이므로 \(\det (\boldsymbol{A})= \pm 1\) 이다.


14.4.2 심플렉틱 군

심플렉틱 군에 대해서는 간단히 소개만 하고 넘어가기로 한다.

정의 14.10 (심플렉틱 군) \(n\) 차원 벡터공간 \(V\) 에 대해 \(B\in \mathcal{A}^2(V)\) 일 때 다음과 같이 정의된 \(Sp(V, B)\) 를 \(B\) 로부터 생성된 심플렉틱 군 (symplectic group) 이라고 한다.

\[ Sp (V, B)=\{ L\in GL(V) : T(Lv, Lw) = T(v, w)\}. \]


명제 14.14 유한차원 벡터공간 \(V\) 와 \(B\in\mathcal{A}^2(\mathbb{F})\) 를 생각하자. \(V\) 의 기저 \(\mathcal{B}\) 에 대해 다음은 동치이다.

 (\(1\)) \(L \in Sp(V, B)\).

 (\(2\)) \(([L]_\mathcal{B})^T[B]_\mathcal{B}([L]_\mathcal{B}) =[B]_\mathcal{B}\).


(증명). \(v,\,v'\in V\) 에 대해 \([v]_\mathcal{B}=\boldsymbol{v},\, [v']_\mathcal{B}=\boldsymbol{v}'\) 이라 하고 \([L]_\mathcal{B}=\boldsymbol{L},\, [B]_\mathcal{B}= \boldsymbol{B}\) 라고 하자. 우선 \(B,\,B'\in \mathcal{Q}_V\) 일 때 \(B (v, v') = B' (v, v') \iff B = B'\) 임은 당연하다. 따라서 \(\boldsymbol{v}^T[B]\boldsymbol{v} = \boldsymbol{v}^T[B']\boldsymbol{v} \iff [B]=[B']\) 를 의미한다.

(\(1 \implies 2\)) \(B(Lv, Lv') = \boldsymbol{v}^T\boldsymbol{L}^T\boldsymbol{BLv}\), \(B (v,v) = \boldsymbol{v}^T\boldsymbol{Bv}\) 이다. 따라서 \(\boldsymbol{L}^T\boldsymbol{BL}= \boldsymbol{B}\) 이다.

(\(2 \implies 1\)) \(\boldsymbol{L}^T\boldsymbol{BL}= \boldsymbol{B}\) 이면 \(\boldsymbol{v}^T\boldsymbol{L}^T\boldsymbol{BLv}\), \(B (v,v) = \boldsymbol{v}^T\boldsymbol{Bv}\) 이므로 \(L\in Sp(V,B_\Psi)\) 이다. \(\square\)


14.5 특별한 직교군

14.5.1 \(O(1, 1)\)

대칭행렬에 대한 직교군 을 생각하자.

정의 14.11 (\(O(1, 1)\)) \(\mathcal{M}_{2 \times 2}(\mathbb{R})\) 에서 정의된 \(\boldsymbol{J} = \begin {bmatrix} 1 & 0 \\ 0 & -1\end{bmatrix}\) 에 대한 직교군 \(O_2^\boldsymbol{J}\) 을 \(O(1, 1)\) 이라고 한다.


보기 14.4 다음은 동치이다.

 (\(1\)) \(\boldsymbol{A} = \begin{bmatrix} x& z \\ y & w\end{bmatrix} \in O(1, 1)\).

 (\(2\)) \(x^2-y^2 = w^2-z^2 = 1,\, xz-yw=0\).
