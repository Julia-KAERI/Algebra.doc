13  몫공간과 몫사상

13.1 아핀 부분집합과 몫공간

정의 13.1 (부분공간의 평행이동) 벡터공간 \(V\) 의 정해진 \(v\in V\) 와 \(V\) 의 부분공간 \(U\) 에 대해 \(V\) 의 부분집합 \(v+U\) 를 다음과 같이 정의한다.

\[ v+U := \{v+u : u\in U\}. \]

이 때 \(v+U\) 는 \(U\) 가 \(v\) 만큼 평행이동했다고 한다.


정의 13.2 (몫공간) \(U\) 가 \(V\) 의 부분공간일 때 몫공간 (quotient space) \(V/U\) 는 \(U\) 가 평행이동한 부분집합의 집합이다. 즉,

\[ V/U = \{v+U: v\in V\} \]

이다.

공간이라는 이름을 붙이는 이유는 뒤에 보이겠지만 적절하게 합과 스칼라곱을 정의하면 \(V/U\) 가 벡터공간이 되기 때문이다.


정리 13.1 (몫공간의 성질) \(v_1,\,v_2 \in V\) 와 \(U\le V\) 에 대해 다음이 성립한다.

  (\(1\)) \(v_1-v_2\in U\) 이면 \(v_1+U = v_2+U\) 이다.

  (\(2\)) \(v_1-v_2\notin U\) 이면 \((v_1 + U)\cap (v_2+U) = \varnothing\) 이다.

(증명). \(v_1 - v_2 = u\in U\) 이면 \(v_1+U = v_2+U\) 임은 자명하다. 이제 \(v_1-v_2\notin U\) 일 때 \(x\in (v_1+U)\cap (v_2+U)\) 라고 하자. \(x=v_1+u_1 = v_2+u_2\) 이므로 \(v_1-v_2 = u_2-u_1 \in U\) 인데 이는 \(v_1-v_2\not\in U\) 인 가정에 모순된다. \(\square\)


정의 13.3 (몫공간에서의 합과 스칼라곱) \(U\) 가 \(\mathbb{F}\)-벡터공간 \(V\) 의 부분공간일 때 \(V/U\) 의 원소들간의 합과 스칼라곱을 다음과 같이 정의한다.

\[ \begin{array}{c} (v + U) + (w + U) = (v + w) +U \\ \lambda (v + U) = \lambda v + U \end{array} \]


명제 13.1 \(V/U\) 는 벡터공간이다.

(증명). 우선 \(v_1,\,v_2\) 가 \(v_1-v _2 \in U\) 이면 \(v_1 + U =v_2 + U\) 임은 쉽게 보일 수 있다. 또한 \(U \in V/U\) 이며 \((v + U) + U = v + U\) 이므로 \(U\) 는 \(V/U\) 의 항등원이다. 나머지 벡터공간의 성질은 쉽게 보일 수 있다. \(\square\)


13.2 몫사상

정의 13.4 (몫사상 (Quotient map)) \(U\) 가 \(\mathbb{F}\)-벡터공간인 \(V\) 의 부분공간일 때 다음과 같이 정의되는 함수 \(\pi_U : V \to V/U\) 를 몫사상(quotient map) 이라 한다.

\[ \pi_U (v) = v + U \]


명제 13.2 몫사상 \(\pi_U\) 는 선형사상이다.

(증명). 몫사상임을 보이기 위해서는 \(v_1,\,v_2 \in V\), \(\lambda \in \mathbb{F}\) 일 때 \(\pi_U (v_1+ \lambda v_2)= \pi_U (v_1) + \lambda \pi_U (v_2)\) 임을 보이면 된다. \[ \begin{aligned} \pi_U (v_1+\lambda v_2) &= v_1 + \lambda v_2 + U = (v_1 + U) + \lambda (v_2 + U) \\ &= (v_1 + U) + (v_2 + \lambda U) \\ &= \pi_U(v_1) + \lambda \pi_U(v_2) \\ \end{aligned} \] \(\square\)


정리 13.2 \(V\) 가 유한차원벡터공간이고 \(U\) 가 \(V\) 의 부분공간일 때

\[ \dim (V/U) = \dim V - \dim U \]

이다.

(증명). \(\ker (\pi_U) = U\) 이다. \(\square\)


정의 13.5 (\(\tilde{T}\)) 선형사상 \(T\in \mathcal{L}(V,\,W)\) 에 대해 함수 \(\tilde{T} : V/\ker (T) \to \text{im}\, T\) 를 다음과 같이 정의한다.

\[ \tilde{T}(v + \ker (T)) = T(v) \]


명제 13.3 \(\tilde{T}\) 는 선형사상이다.

(증명). 우선 \(\ker (T)\) 가 \(V\) 의 부분공간이므로 \(V/\ker (T)\) 가 정의된다. \(\tilde{T}\) 가 잘 정의된 함수임을 보이려면 \(v_1 +\ker (T) = v_2 + \ker (T)\) 인 \(v_1,\,v_2\) 에 대해 \(T(v_1) = T(v_2)\) 임을 보이면 되는데 이는 쉽게 보일 수 있다. 또한 \(\tilde{T}\) 가 선형사상임을 보이는 것도 쉽다. \(\square\)


명제 13.4 \(\tilde{T}\) 에 대해 다음이 성립한다.

  1. \(\tilde{T}\) 는 단사이다.

  2. \(\text{im}\, \tilde{T} = \text{im} \, T\)

  3. \(V/\ker (T)\) 는 \(\text{im}\, T\) 와 동형이다.

(증명은 쉽다.)
